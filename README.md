# Portafolio Profesional - José Azócar

Este repositorio contiene el código fuente del portafolio personal de **José Azócar**, accesible en [joseazocar.pro](https://joseazocar.pro). El sitio ha evolucionado para convertirse en un centro de recursos profesional que destaca su trayectoria en ingeniería, ciberseguridad y desarrollo de software, presentando una interfaz moderna, optimizada y de alto rendimiento.

El proyecto no solo sirve como carta de presentación, sino como un catálogo interactivo de soluciones técnicas, asesorías especializadas y proyectos de código abierto.

## 🚀 Características Actualizadas

* **Arquitectura Modular**: Estructura organizada que separa claramente las secciones de Experiencia, Formación, Proyectos y Servicios.
* **Sección de Asesorías IT**: Espacio dedicado a servicios profesionales en infraestructura, seguridad y optimización de sistemas.
* **Showcase de Proyectos (Challenges)**: Galería dinámica de desafíos técnicos superados, con integración directa a repositorios de GitHub.
* **Optimización SEO y Performance**: Configurado para tiempos de carga mínimos y mejor visibilidad en motores de búsqueda.
* **Interfaz Adaptativa (Responsive)**: Experiencia de usuario fluida en dispositivos móviles, tablets y ordenadores de escritorio.
* **Seguridad Implementada**: Uso de cabeceras de seguridad y mejores prácticas de protección de datos.

## 🛠️ Tecnologías y Metodologías

El stack tecnológico ha sido refinado para garantizar escalabilidad y mantenibilidad:

* **Frontend**: HTML5 Semántico, CSS3 avanzado (Flexbox/Grid), JavaScript (ES6+).
* **Estilización y Animación**: 
    * Metodología **BEM** (Block Element Modifier) para un CSS escalable.
    * **Animate On Scroll (AOS)** para interactividad visual.
    * **Font Awesome** & **Google Fonts** para tipografía e iconografía.
* **Herramientas de Desarrollo**: Git para control de versiones y despliegue continuo (CI/CD).
* **Despliegue**: Alojamiento profesional con soporte para SSL/TLS, garantizando una conexión segura vía HTTPS.

## ⚙️ Instalación y Configuración Local

Si deseas explorar el código o ejecutar una copia local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/azocarone/portafolio.git
    ```
2.  **Navegar al directorio:**
    ```bash
    cd portafolio
    ```
3.  **Ejecución:**
    Simplemente abre el archivo `index.html` en tu navegador o utiliza una extensión como **Live Server** en VS Code para previsualizar los cambios en tiempo real.
